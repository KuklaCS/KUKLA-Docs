The DWC-8B is internally a "downsized" DWC-7B,

so in terms of Fieldbus communication, especially data frames, GSDML,GSD,EDS it is 1:1 like a DWC-7B

So, see section DWC-7B section Fieldbus, Everything is also usable for DWC-8B, even if only DWC-7 is mentioned !!!!
